# Job-Analysis-2026
# 📊 Data Analyst Job Market Analysis

## Overview

This project explores the data analyst job market using Python.
The goal is to understand which skills are in demand, how salaries vary, and what skills are worth learning.

The dataset includes job roles, salaries, locations, and required skills.

---

## Questions I Answered

* What skills are most in demand for data roles?
* How do skills trend over time?
* Which skills pay the most?
* What are the best skills to learn (high demand + high pay)?

---

## Tools Used

* **Python**
* **Pandas** – data analysis
* **Matplotlib & Seaborn** – data visualization
* **Jupyter Notebook** – running code and analysis
* **Git & GitHub** – version control

---

## What I Did

* Cleaned and prepared the dataset
* Analyzed skill demand and trends
* Compared salaries across roles
* Found high-paying and high-demand skills
* Created charts to visualize insights

---

## Key Insights

* **SQL** and **Excel** are the most in-demand skills
* **Python** is highly valuable across all data roles
* Specialized skills can lead to higher salaries
* A mix of basic + advanced skills is best for career growth

---

## What I Learned

* How to clean and analyze real-world data
* How to create meaningful visualizations
* How to connect skills with job market demand

---

## Conclusion

This project helped me better understand the data analyst job market and the skills needed to succeed.
It’s a strong starting point for anyone interested in data analytics.
